# Tone and Style Guidelines for Natural Chinese Writing

## Register Mapping

| Source Register | Target Tone | Key Adjustments |
|----------------|-------------|-----------------|
| Academic / formal | Authoritative but readable | Shorten sentences, reduce passive voice ("被...所..."), keep terminology |
| Business / professional | Clear and direct | Remove bureaucratic padding ("鉴于...特此..."), use active voice |
| Casual / blog | Conversational | Add colloquial expressions, use shorter paragraphs, allow incomplete sentences |
| Technical documentation | Precise and scannable | Keep technical terms, improve heading hierarchy, tighten explanations |

## Common Substitutions

### Overused Transitions (Replace or Remove)

- "首先...其次...再次...最后" -> Use varied connectives or simply natural paragraph flow
- "值得注意的是" -> Drop entirely, or use "注意" if emphasis is truly needed
- "不可否认" -> Usually removable; the statement that follows should stand on its own
- "众所周知" -> Remove; if it were truly well-known, no need to say so

### Filler Phrases (Usually Remove)

- "在当今社会" / "在这个时代" -> Remove unless historical contrast is the point
- "从某种意义上来说" -> Remove; commit to the statement
- "事实上" / "实际上" -> Keep only when genuinely contrasting with a prior misconception

## Rhythm Guidelines

- Aim for a mix of sentence lengths: 5-15 characters (short), 15-30 (medium), 30-50 (long)
- Avoid more than two long sentences in succession
- Use sentence fragments deliberately for emphasis in casual registers
- End paragraphs with shorter, punchier sentences when possible
